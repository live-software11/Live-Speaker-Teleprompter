# Italiano

## Cos'è Live Speaker Teleprompter

**Live Speaker Teleprompter** è un'applicazione desktop per **teleprompter** professionale: mostra il testo dello script su schermo con velocità e stile configurabili, ideale per speaker, presentatori e registrazioni video.

L'app permette di:

- Visualizzare lo **script** con scorrimento fluido e velocità regolabile
- Trasmettere su **display esterni** (monitor secondari)
- Controllare da **Stream Deck** tramite Companion (play, pause, velocità)
- Personalizzare **font, dimensioni, colori** per massima leggibilità

## Per chi è

- Speaker e presentatori
- Tecnici AV per eventi live
- Registrazioni video e podcast
- Chi deve leggere script senza perdere il contatto visivo con la telecamera

## Funzionalità principali

- **Scorrimento testo** — Velocità regolabile, direzione, loop
- **Trasmissione display** — Finestre su monitor secondari
- **Companion** — Integrazione Stream Deck per controllo remoto
- **Personalizzazione** — Font, dimensione, colore testo e sfondo
- **i18n** — Italiano e Inglese

## Requisiti

- **Sistema:** Windows 10/11 (64 bit)
- **WebView2:** Incluso in Windows 11
- **Companion:** Opzionale, per controllo da Stream Deck

## Come funziona

1. Incolla o carica lo script
2. Regola velocità e stile (font, colori)
3. Assegna lo schermo per la trasmissione
4. Avvia lo scorrimento; controlla da Stream Deck se configurato

# English

## What is Live Speaker Teleprompter

**Live Speaker Teleprompter** is a desktop application for a **professional teleprompter**: it displays script text on screen with configurable speed and style, ideal for speakers, presenters, and video recordings.

The app lets you:

- Display the **script** with smooth scrolling and adjustable speed
- Transmit to **external displays** (secondary monitors)
- Control from **Stream Deck** via Companion (play, pause, speed)
- Customize **font, size, colors** for maximum readability

## Who it's for

- Speakers and presenters
- AV technicians for live events
- Video and podcast recordings
- Anyone who needs to read scripts without losing eye contact with the camera

## Main features

- **Text scrolling** — Adjustable speed, direction, loop
- **Display transmission** — Windows on secondary monitors
- **Companion** — Stream Deck integration for remote control
- **Customization** — Font, size, text and background color
- **i18n** — Italian and English

## Requirements

- **System:** Windows 10/11 (64 bit)
- **WebView2:** Included in Windows 11
- **Companion:** Optional, for Stream Deck control

## How it works

1. Paste or load the script
2. Adjust speed and style (font, colors)
3. Assign the screen for transmission
4. Start scrolling; control from Stream Deck if configured
